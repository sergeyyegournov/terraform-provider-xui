# xui_shadowsocks_client (Resource)

Shadowsocks user (client) on an existing 3x-ui inbound. Managed via `/panel/api/clients/*` (`add`, `get/:email`, `update/:email`, `del/:email`). Requires 3x-ui v3+ with the dedicated client management API.

The panel generates the client `password` when it is omitted in Terraform; the value is stored in state after create.

## Example Usage

```terraform
resource "xui_shadowsocks_client" "example" {
  inbound_id = xui_inbound.ss.id
  email      = "client@example.com"
}
```
